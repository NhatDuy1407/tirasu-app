Tirasu

Multi-Platform Video Upload Tool

Effective date: March 5, 2026

Tirasu is a personal, non-commercial desktop application developed and used exclusively by its developer. It has no public users, customers, or end-user accounts. This Privacy Policy is provided to meet third-party API registration requirements (e.g., TikTok for Developers) and to document how the App handles data.

1. Who Uses This App

Tirasu is used solely by the individual developer who built it. It is not distributed, published, or made available to any other person. There is no user registration, no user accounts, and no collection of data belonging to anyone other than the developer themselves.

2. Data Handled by the App

2.1 Developer-Provided Data

  • Video files and metadata – titles, descriptions, tags, thumbnails, and scheduling details entered by the developer.
  • API keys – e.g., a Google Gemini API key owned by the developer, stored locally on their device.
  • Preferences – custom AI prompt templates, default tags, and scheduling times.

2.2 OAuth Tokens from Platform Authorization

When the developer connects their own social media account, the platform's OAuth flow returns an access token. The following are stored locally on the developer's device only:

  • OAuth access token and refresh token.
  • Token expiration timestamps.
  • The developer's own display name and account ID on that platform.
  • Profile avatar URL (for display only).

2.3 TikTok API Permissions

Tirasu requests only the minimum TikTok permissions required to upload videos to the developer's own account:

Permission ScopePurpose
video.uploadUpload video files to the developer's TikTok account.
video.publishPublish or schedule uploaded videos on the developer's account.

Tirasu does not access, read, or store TikTok followers, following lists, messages, analytics, or any data unrelated to performing uploads.

2.4 No Telemetry or Analytics

Tirasu does not collect device identifiers, IP addresses, usage analytics, or crash reports. No data is sent to any analytics service or the developer's own servers.

3. How Data Is Used

  • To authenticate the developer's own social media accounts and perform uploads on their behalf.
  • To display account status and upload history within the App.
  • To send video files and metadata to the platforms the developer selects at upload time.
  • To generate metadata suggestions via Google Gemini using the developer's own API key.
  • To persist preferences across sessions.

4. Data Storage

All data is stored in a local SQLite database on the developer's own device. No cloud servers, remote databases, or third-party storage services are used by Tirasu. The only external connections made by the App are:

  • To platform APIs (TikTok, YouTube, etc.) to perform authorized uploads.
  • To the Google Gemini API for AI metadata generation, using the developer's own API key.

5. Data Sharing

No data is sold, shared, or disclosed to any third party. Data flows only between the App and the platforms explicitly authorized by the developer for the purpose of uploading their own content.

6. Data Deletion

All locally stored data can be removed at any time by disconnecting accounts within the App or by uninstalling the application, which removes the local database entirely.

7. Third-Party Privacy Policies

When data is transmitted to connected platforms, it becomes subject to their own privacy policies:

  • TikTok Privacy Policy
  • Google / YouTube Privacy Policy
  • Meta (Instagram / Facebook) Privacy Policy

8. Contact

For any questions about this Privacy Policy, contact: support@tirasu.app
